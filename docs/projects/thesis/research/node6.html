Delay-and-Sum Beamforming

Now that we have removed the echo from the near end signal, it's time to think of other ways to improve the quality of speech. One way to do this is to increase the number of microphones (and call it an array). Just like increasing the temporal sample rate tells us more about the signal, increasing the number of spatial samples tells us more as well.
